Hábitos e Perfomance Acadêmica
📖 Sobre o Projeto
Este projeto tem como objetivo analisar como os hábitos dos estudantes influenciam seu desempenho acadêmico. Foram estudados fatores como horas de estudo, qualidade do sono, prática de atividades físicas, uso de redes sociais e frequência às aulas.

A análise busca identificar padrões e relações entre esses hábitos e as notas obtidas pelos estudantes.

---

🎯 Objetivos
Investigar os principais hábitos dos estudantes.
Analisar a relação entre hábitos e desempenho acadêmico.
Identificar fatores que contribuem para melhores resultados.
Criar visualizações para facilitar a interpretação dos dados.

---

📊 Base de Dados
O conjunto de dados contém informações sobre:

Horas de estudo por dia
Horas de sono
Frequência às aulas
Tempo gasto em redes sociais
Atividades físicas
Nível de estresse
Nota final ou desempenho acadêmico
